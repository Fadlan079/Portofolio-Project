Tentang Saya

Saya adalah lulusan baru Ilmu Komputer dengan minat dan keahlian di bidang pengembangan perangkat lunak, khususnya web development. Selama studi, saya mengerjakan berbagai proyek akademik dan magang yang membantu saya memahami konsep pemrograman serta praktik nyata dalam membangun aplikasi.

Saya menguasai Python, PHP, JavaScript, HTML, dan CSS, serta memiliki pemahaman tentang pengembangan web yang semantik dan responsif. Selain itu, saya berpengalaman menggunakan Laragon untuk server-side development. Dengan kombinasi latar belakang akademik dan pengalaman praktis tersebut, saya siap berkontribusi serta berkembang sebagai profesional di bidang software development.
